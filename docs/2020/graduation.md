# 毕业

## 你好暨阳，再见暨阳

四年前，迈进这个叫暨阳的学校。四年后，走出这个叫暨阳的学校，四年不长不短，收获了许多，也学到了很多，也交到了很多朋友。

对于这个学校，爱过也恨过，爱是因为，无论如何，这都是我未来的母校，我是从这里出来的，哪怕学校再差，也是以前收留过我的地方，恨是因为，对于学校的一些制度的不满，对于一些不公平的不满。但是这些点点滴滴却又给我留下了很深的印象。
